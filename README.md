# ⛺ ふぁみキャン△

家族のキャンプを計画・記録するアプリです。メンバー・やること・買い出しリスト・写真をひとまとめに管理できます。

**URL:** https://fami-camp.vercel.app/

---

## 機能

### キャンプ管理
- **リスト画面**：予定中・進行中のキャンプ一覧 ＋ 過去の記録一覧
- **詳細画面**：キャンプの全情報を確認、完了処理、写真の追加
- **フォーム画面**：新規キャンプの作成・編集

### キャンプ情報
各キャンプには以下の情報を登録できます：

| 項目 | 内容 |
|---|---|
| タイトル | 必須。例：「夏キャン！2025」 |
| キャンプ場名 | 例：ほったらかしキャンプ場 |
| 住所 | 天気取得などに使用 |
| 開始日・終了日 | キャンプ期間 |
| 集合場所・時間 | 集合情報 |
| メンバー | 名前＋人数。よく使うグループを保存可能 |
| やりたいこと | 料理・遊び・その他でカテゴリ分け |
| 買い出しリスト | 品目・担当者 |
| メモ・再訪フラグ | 次回への引き継ぎメモ |
| 写真 | コメント付きで複数枚追加可能 |

### 完了処理
キャンプ終了時に「記録として保存」でき、5段階の「またここに行きたい度」を評価として残せます。

### よく使うグループ
一緒に行くメンバーの組み合わせをグループとして保存し、次回のキャンプでワンタップで追加できます。

## ファイル構成

```
fami-camp/
├── index.html     # アプリ本体（リスト・詳細・フォーム画面）
├── styles.css     # スタイル
├── script.js      # アプリロジック
├── config.json    # Supabase接続情報（gitignore推奨）
└── vercel.json    # Vercelデプロイ設定
```

---

## Supabase接続

`config.json` に Supabase の接続情報を記述します：

```json
{
  "supabaseUrl": "https://xxxx.supabase.co",
  "supabaseKey": "your-anon-key"
}
```

### テーブル構成

| テーブル | 内容 |
|---|---|
| `camps` | キャンプ基本情報（タイトル・キャンプ場・日程・ステータス等） |
| `camp_members` | キャンプ参加メンバー |
| `camp_todos` | やりたいことリスト（カテゴリ・達成フラグ） |
| `camp_shopping` | 買い出しリスト（品目・担当者・購入済みフラグ） |
| `camp_photos` | キャンプ写真（Supabase Storage URL・コメント） |
| `favorite_groups` | よく使うメンバーグループ |
